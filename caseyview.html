Video Switcher Product Manual

Product Specifications

Video I/O & Processing

  • **Video Inputs:** 8 x 3G-SDI, 4 x HDMI 2.0, 2 x NDI/SRT IP
  • **Video Outputs:** 4 x 3G-SDI, 2 x HDMI 2.0 (Programmable)
  • **Supported Formats:** HD (1080p60, 1080i60, 720p60), 4K UHD (2160p60)
  • **Keying:** 2 x Upstream, 2 x Downstream
  • **Transitions:** Cut, Mix, Wipe, DVE

Audio I/O

  • **Audio Embedding/De-embedding:** Embedded on all SDI and HDMI inputs.
  • **Analog Audio:** 2 x XLR/TRS Combo Inputs.
  • **Audio Mixing:** 2-channel embedded audio mixer.

Control & Networking

  • **Physical Control:** Integrated T-Bar, backlit buttons, joystick.
  • **Software Control:** Ethernet port for PC/Mac.
  • **Tally:** 15-pin D-sub port.

Physical & Electrical

  • **Dimensions:** 2U Rack Mount Chassis, 19" x 3.5" x 12" (W x H x D)
  • **Weight:** 15 lbs (6.8 kg)
  • **Power Requirements:** 100-240V AC, 50/60 Hz, 85W Max

Quick Start Guide

1.
Physical Installation: Secure in a standard 19-inch equipment rack. Connect the included power cable to a standard AC power outlet.
2.
Input Connections: Connect video sources (cameras, laptops) to the **SDI** and **HDMI** inputs.
3.
Output Connections: Connect video monitors or streaming encoders to the **SDI** and **HDMI** outputs.
4.
Power On: Press the **POWER** button. The multi-viewer outputs will automatically display a default layout of all connected sources.
5.
Basic Switching: Use the **Program** and **Preview** rows of buttons to select sources. Use the **T-Bar** for a smooth transition or press **CUT** for an instantaneous switch.

Operational Controls and Menus

Integrated Control Panel

The hardware panel features **Program** and **Preview** rows for source selection. The **T-Bar** controls transition speed, while the **CUT** button performs an instant switch. A joystick is used for DVE and PTZ camera control.

Software Menu (PC/Mac)

The dedicated software provides full control over advanced routing, detailed keyer adjustments, macros, and the audio mixer.

Troubleshooting

Issue Probable Cause Corrective Action
No Video Output Output is set to the wrong source or incorrect format. Check the Program row and use CUT. Verify output format compatibility in the software menu.
Video Signal is Black Input signal is not valid. Verify the source is powered on and sending a valid signal. Check the multi-viewer.
Buttons/T-Bar Unresponsive Unit has frozen or software control is overriding hardware. Perform a hard restart or disconnect the Ethernet cable and restart.
No Audio Embedded audio is not selected or analog audio is not connected. Ensure the active audio source is routed to the main mix in the software, and check analog cable connections.

Block Diagram

The diagram follows a left-to-right flow: from inputs, through the central processing core, to the outputs.

Video Signal Path

  • **Inputs:** SDI, HDMI, and IP signals feed into a **Video Scaler**.
  • **Processing Core:** The core engine contains the **Program Bus** and **Preview Bus** connected by the **T-Bar/Transition Engine**. It also handles **Upstream/Downstream Keyers**.
  • **Outputs:** Programmable SDI and HDMI outputs are assigned from the various buses.

Audio Signal Path

  • **De-embedding:** Audio is extracted from SDI and HDMI inputs.
  • **Mixing:** Analog inputs and de-embedded audio are sent to a central mixer.
  • **Embedding:** The final mixed audio is re-embedded into the main video output.

Control Path

  • **Ethernet:** Provides control via PC/Mac software.
  • **Tally:** Sends signals to cameras.
